# Lab 6: Token Parse

In this lab, we write a program which contains a function to parse a single line of input and and prints out the individual tokens.

## Files: 

* repl.c

## Input and Output Examples:

./parse "cd ."
```
cd
.
```

./parse "ls -l"
```
ls
-l
```
./parse "cat compile.py | head 2"
```
cat
compile.py
|
head
2
```
